### 算法评价

- 算法的正确性——调试，人工证明
- 算法的有效性——算法的时间复杂性、算法的空间复杂性

T(n) = O(f(n))
- 有效：O(1)、O(logn)、O(n)、O(n logn)、O(n^2)、O(n^3)、O(n^c)(c是常数，多项式阶)
- 无效：O(2 ^ n)、O(n!)

时间复杂性：

- 最坏情况：具有相同输入数据量的不同输入数据，算法时间用量的最大值——T_w(n)
- 平均情况：~，算法耗用时间的**平均值**——T_E(n)

空间复杂性：S(n)

### 算法选用原则

- 实现的难易程度
- 算法的使用次数
- 算法运行环境
